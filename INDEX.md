### 2. **popup/tooltips.js** — Логика Tooltips
```
Размер: ~4KB
Функция: Управление показом/скрытием подсказок
Содержит:
  - TooltipManager модуль
  - Smart positioning логика
  - Обработчики событий (hover, focus, blur)
  - Динамическое отслеживание элементов
```

**API**:
```javascript
TooltipManager.init()        // Инициализация
TooltipManager.show(el)      // Показать
TooltipManager.hide()        // Скрыть
TooltipManager.attach(el)    // Подключить к элементу
```

**Конфигурация**:
```javascript
const SHOW_DELAY = 200;      // мс перед появлением
const HIDE_DELAY = 100;      // мс после скрытия
```
