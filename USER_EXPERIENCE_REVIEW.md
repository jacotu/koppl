# Koppl — User Experience Review

**Reviewer perspective**: Non-technical user who wants to customize fonts on websites
**Test date**: 2026-03-17
**Overall rating**: 8/10

---

## 🎯 First Impression

**Opening the extension**
- ✅ Clean, modern interface with clear sections
- ✅ Four tabs (Inspect, Upload, Apply, Sites) are logically organized
- ⚠️ Lots of buttons and options — potentially overwhelming for first-time user
- ✅ Dark/light theme toggle is a nice touch

**Initial UX**: 8/10 — Professional look, but dense

---

## 📥 Getting Started: Upload Flow

**Scenario**: User wants to use a custom font

| Step | Feedback | Rating |
|------|----------|--------|
| 1. Click "Upload" tab | Clear destination, obvious next step | ✅ |
| 2. Drag-and-drop or browse | Works as expected; both methods available | ✅ |
| 3. Click "Save fonts" button | Straightforward, but no progress indicator while saving | ⚠️ |
| 4. Font appears in list | Good; shows font name and file size | ✅ |

**Result**: Easy once you find the Upload tab. Progress bar at bottom helps.
**Rating**: 8/10

---

## 🎨 Main Feature: Applying Fonts

**Scenario**: User wants to change font on current website

**The good:**
- ✅ Font dropdown with search — very fast to find fonts
- ✅ Live preview with slider — instantly see size changes
- ✅ Preview text is meaningful ("Sphinx of black quartz...")
- ✅ "Apply" button is prominent and obvious
- ✅ Works immediately after clicking

**The not-so-good:**
- ⚠️ Typography options are hidden in accordions — user might miss them
- ⚠️ "Apply to" selector (Whole page vs Headings vs Custom) might be confusing for newcomers
- ⚠️ "Text-only mode" and "Relax containers" — unclear what these do without hovering
- ⚠️ OpenType features section is too technical for average user

**Result**: Core feature works great. Advanced options are well-organized but confusing.
**Rating**: 8/10

---

## ⚙️ Advanced Features

**Presets** (Save typography settings)
- ✅ Easy to save, easy to reuse
- ⚠️ No indication of what's saved (would benefit from preview)

**Font Rules** (Different fonts for different elements)
- ⚠️ Requires CSS knowledge or guessing
- ⚠️ "Relax containers" option needed — why not automatic?

**Accessibility Checker**
- ✅ Great feature for power users
- ⚠️ Overkill for someone just wanting to change a font
- ✅ Color inputs work well

**Inspect Tab** (See what fonts a site uses)
- ✅ Useful for curious users
- ⚠️ Needs a "Scan page" button click — not automatic

**Rating**: 7/10 — Features are powerful but require learning curve

---

## 💾 Sites Management

**Viewing saved configurations**
- ✅ Shows all sites where fonts were applied
- ✅ Search and sort work well
- ✅ Export/import feature is gold for backup
- ⚠️ No way to quickly reset all sites at once
- ⚠️ No "sync across devices" (expected limitation for local extension)

**Rating**: 8/10

---

## 🎯 Core Workflow Analysis

### Ideal user journey (Happy path):
1. Open extension
2. Go to Upload tab
3. Upload a font
4. Go to Apply tab
5. Select font
6. Click Apply
7. Done ✅

**Time to complete**: ~10 seconds
**Difficulty**: Very easy

### Reality:
- First-time users might not find Upload tab immediately (it's second)
- Defaulting to "Inspect" tab is unusual — most extensions show main feature first
- Should show a "Quick start" hint on first use

---

## 🐛 Potential Issues

| Issue | Severity | Impact |
|-------|----------|--------|
| No visual feedback while saving fonts | Medium | User doesn't know if it worked |
| Accordions hidden by default | Medium | Users miss typography options |
| "Apply to" modes need explanation | Low | Trial-and-error works |
| No "undo" for Apply action | Low | Requires manual reset |
| Can't disable extension per-tab (only per-site) | Low | Slight inconvenience |

---

## 🎯 Compared to Expectations

**What users expect from a font extension:**
- ✅ Upload fonts
- ✅ Apply to websites
- ✅ See results immediately
- ⚠️ Simple interface (Koppl is feature-rich, not simple)
- ✅ Save preferences per-site
- ⚠️ One-click operation (requires 2-3 clicks)

---

## ✨ Strengths (User Perspective)

1. **It works** — No broken features or crashes
2. **Fast** — No noticeable lag or slowness
3. **Privacy** — Fonts stay on my computer, not uploaded anywhere
4. **Powerful** — Advanced users can do lots
5. **Free** — No premium tier or ads
6. **Keyboard shortcut** — Ctrl+Shift+F toggle is fast
7. **Live preview** — See changes before applying

---

## 😞 Weaknesses (User Perspective)

1. **Learning curve** — Too many options for beginners
2. **Default tab** — Inspect first, not Upload or Apply
3. **Unclear terminology** — "Text-only mode", "Relax containers", "Font rules"
4. **No visual indicators** — Empty page when no fonts loaded
5. **No quick tutorial** — First-time experience not smooth
6. **Too many accordions** — Easy to hide important features
7. **No favorites system** — Hard to track which fonts I like

## 🎯 Final User Rating: **8/10**

### Summary

**Koppl is a solid, professional font extension that does what it promises.**

**Best for:**
- Users who want fine-grained control over typography
- People with dyslexia or visual impairments
- Web designers testing fonts
- Power users who like customization

**Not ideal for:**
- Someone who just wants to "make fonts bigger"
- Users who don't like learning new tools
- People on mobile (no mobile version)

### Quick Recommendation
If you're comfortable with settings and options, you'll love this. If you want a one-button solution, look elsewhere.

### What Would Make It 9/10+
1. **Onboarding tour** on first use
2. **Favorites system** for quick font access
3. **Simpler "quick mode"** with just 3 options (font, size, apply)
4. **Default tab** should be Apply, not Inspect
5. **Tooltips** on all unclear buttons
6. **One-click examples** ("Make it bigger", "Use dyslexia font")

---

**Overall Assessment**: A powerful, well-built tool that feels professional but requires learning. Worth the 5-minute investment to understand all features. Highly recommended for anyone who customizes their browsing experience.
