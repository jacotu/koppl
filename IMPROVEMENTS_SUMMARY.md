## 🚀 Как Использовать

### Для Пользователя
1. **Прочитай README.md** — узнаешь что это и как начать
2. **Открой расширение** — каждый элемент имеет подсказку на наведение
3. **Наводи мышку** — подсказки появляются за 200мс
4. **Ищи помощь** — если что-то непонятно, смотри TOOLTIPS_GUIDE.md

### Для Разработчика
1. **popup-tooltips.css** — CSS для внешнего вида
2. **popup/tooltips.js** — Основная логика (можно настроить delays)
3. **popup.html** — Элементы с `data-tooltip` атрибутами

## ⚙️ Технические Детали

### Tooltips
- **Тип**: CSS + Vanilla JavaScript (без зависимостей)
- **Размер**: ~3KB CSS + ~4KB JS
- **Производительность**: Оптимизировано, минимум перерисовок
- **Поддержка**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

### Дизайн
- **Тема**: Light/Dark mode aware
- **Позиционирование**: Автоматическое (smart positioning)
- **Анимация**: Smooth fade-in 200ms
- **Мобильная**: Отключено на экранах < 480px
